# \# Advanced HTML \& CSS Project

# 

# \## Overview

# This project is part of the ALX HTML \& CSS advanced concepts. The goal is to recreate a webpage based on a Figma design using HTML5 and CSS3. Focus is on styling, layout, and responsiveness.

# 

# \## Features

# \- Header with logo and navigation menu

# \- Responsive hero section

# \- Feature sections styled according to Figma

# \- Footer with copyright information

# \- Mobile-friendly design

# 

# \## Resources

# \- Figma design file: \[Link to Figma](#)

# \- Downloaded images used for logo and content

# \- Fonts used: Source Sans Pro, Spin Cycle OT

# 

# \## How to run

# 1\. Open `index.html` in any modern browser.

# 2\. Ensure the `styles.css` and image files are in the same directory as `index.html`.
